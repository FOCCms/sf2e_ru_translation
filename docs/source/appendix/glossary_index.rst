.. _trait--weapon--Area:

Область (Area)
""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

- взрыв
- конус

**Источник**: Starfinder field test #1 pg. 8

Это оружие может стрелять только с использованием действия "Огонь по области (Area Fire)".

.. rst-class:: description
.. _item-action--Area-Fire:

Огонь по области (Area Fire) |д-2|
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

- область
- атака

----------

Вы нацеливаетесь на каждое существо в обозначенной области на расстоянии, равном шагу дистанции оружия (для конуса или линии) или указанному радиусу взрыва (для взрыва). 
Для взрыва вы можете расположить центральную точку в любом месте в пределах первого шага дистанции. 
Любые существа в этой области должны преуспеть в базовом спасброске Рефлексов против сложности КС вашего класса плюс значение признака :ref:`trait--weapon--Tracking` оружия (вы не бросаете бросок атаки). 
Этот урон является уроном по области. 
Существа, которые критически провалили этот спасбросок, подвергаются эффектам, возникающим при критическом ударе этим оружием, включая эффект критической специализации оружия.

.. _trait--weapon--Tracking:

Отслеживающее (Tracking)
""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""""

**Источник**: Starfinder field test #1 pg. 9

Это оружие было разработано с несколькими интегрированными системами наведения, стабилизации и самонаведения. 
Броски атаки этим оружием получают бонус предмета, равный указанному значению.
