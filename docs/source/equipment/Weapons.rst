Характеристики оружия (Weapon Statistics)
--------------------------------------------------------------------------------------------------------


.. _ch6--Damage:

Урон (Damage)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Эта запись указывает кость урона оружия и тип урона, который оно наносит: 'Д' для дробящего, 'К' для колющего, или 'Р' для рубящего. 
Некоторое оружие также наносит другие типы урона: 'Ки' для кислотного, 'Х' для урона холодом, 'Э' для электрического, 'О' для огненного, 'П' для психического, 'Я' для урона ядом и 'З' для звукового.

.. _ch6--Weapon--Hands:

Руки (Hands)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Многорукие персонажи: персонажи, у которых более двух рук, например касаты, могут держать больше предметов и оружия, чем обычно ожидается. 
Выполнение действий несколькими парами рук одновременно является сложной задачей и не может быть выполнено без интенсивной подготовки. 
Вы должны назначить пару рук активными руками. 
Вы можете изменить это обозначение с одной пары рук на другую, выполнив действие "Переключить активные руки (Switch Active Hands)", которое является действием Взаимодействия. 
Вы можете атаковать только оружием, которое находится в ваших активных руках.


.. _ch6--Weapon--Capacity:

Ёмкость (Capacity)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

В большинстве видов оружия используются аккумуляторы или магазины, позволяющие производить несколько выстрелов. 
Ёмкость оружия показывает, сколько боеприпасов оно может вместить. 
Аккумуляторное представляют собой оружие с аккумуляторным питанием и аккумулятор максимальной емкости, который оно может вместить. 
Вместо этого во многих огнемётах используется топливо, которое продается в баллонах и работает как аккумуляторы. 
В большинстве огнестрельного оружия используются магазины, вмещающие максимальное количество патронов. 
В зависимости от оружия патронами могут быть болты, дротики, стрелы, патроны или другие боеприпасы. 
В большинстве огнестрельного оружия используются патроны, а в большинстве инъекционного оружия используются дротики. 
Магазины приобретаются индивидуально, по стоимости 1 магазин за кредит. 
Обратите внимание, что это отличается от характеристики `вместимости <https://pf2e-ru-translation.readthedocs.io/ru/latest/appendix/glossary_index.html#trait-weapon-capacity>`_, представленной на странице 150 Pathfinder Guns & Gears.

.. _ch6--Weapon--Usage:

Расход (Usage)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Показывает, сколько боеприпасов расходуется при каждом выстреле, который вы наносите оружием. 
Каждый раз, когда из оружия производится выстрел, его боезапас уменьшается на указанное число.

.. _ch6--Weapon--Usage:


Улучшения (Upgrades)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Оружие можно персонализировать с помощью улучшений, в том числе технологических обвесов и сопряжений. 
В этом поле указано, сколько улучшений может использовать оружие.

Эффекты критической специализации (Critical Specialization Effects)
-----------------------------------------------------------------------------------------------------------------

Ниже приведены эффекты критической специализации для наших новых групп оружия.

.. _weapon--crit-spec--Laser:

Лазер (Laser)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Цель должна преуспеть в спасброске Стойкости против КС вашего класса, иначе она будет :c_dazzled:`ослеплена` до начала вашего следующего хода.

.. _weapon--crit-spec--Projectile:

Снаряд (Projectile)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

Цель должна преуспеть в спасброске Стойкости против КС вашего класса, иначе она будет :c_slowed:`замедлена 1` до конца вашего следующего хода.
