.. _action--Automatic-Fire:

Автоматический огонь (Automatic Fire) |д-2|
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

**Требования**: Оружие, которым вы атакуете обладает свойством :w_automatic:`aвтоматическое`

----------

Вы поражаете каждое существо в конусе на расстоянии, равном половине шага дистанции оружия. 
Любые существа в этой области должны преуспеть в базовом спасброске Рефлексов против сложности вашего класса плюс значение признака отслеживания оружия (вы не бросаете бросок атаки). 
Этот урон является уроном по области. 
Существа, которые критически провалили этот спасбросок, подвергаются эффектам, возникающим при критическом ударе этим оружием, включая эффект критической специализации оружия. 
Автоматический огонь использует половину максимальной ёмкости оружия.
